# TP API Node.js + MongoDB

API REST de gestion d'étudiants développée avec Node.js, Express et MongoDB.

## Installation

git clone https://github.com/syrine-ammar/tp-api-nodejs.git
cd tp-api-nodejs
npm install

## Lancement

npm run dev
